The Strategy
Text Structure & Purpose

Function over topic.

The strat

  1. Find the main idea — what's the whole text about?
  2. Identify the role of each part — setup, twist, conclusion
  3. Match to a function verb — what is this part doing?

Structures you'll see

chronologicalcause → effectcompare / contrastproblem → solutionclaim → evidence

Function verbs

introduceillustratequalifyrefuteconcludesummarize

Trap answers

  • Describe the topic, not the function
  • Repeat passage details without naming the role
  • Misidentify which structure is being used
