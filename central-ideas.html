Central Ideas & Details

What the text actually says.

The strat

  1. Read the whole passage — don't skim
  2. Ask yourself: what's the ONE thing the author wants me to know?
  3. Watch for repetition — what gets mentioned multiple times is usually the main idea
  4. Eliminate choices that focus on details, contradict the text, or add ideas not in it

Sub-types you'll see

Main ideaAuthor's purposeWhat a character feelsWhat a study foundWhat a question is being answered

Quick test for main idea

The main idea = what the WHOLE passage is about. If a choice only covers half the passage, or focuses on one detail, it's not the main idea.

Trap answers

  • True but partial — accurate detail, but not the main point
  • Adds info the passage never mentions
  • Opposite of the passage — wrong feeling, wrong direction
  • Too broad — describes the topic, not the specific idea
